How it works

🤖 Android

Vibration API — duration & patterns are fully supported.

iOS Safari

Vibration API is unavailable. Falls back to a hidden switch-toggle trick — all patterns feel the same (browser limitation).

Basic

On iOS, these all feel identical — the duration is passed to the Vibration API but iOS doesn't support it.

Pattern Android only

On iOS, patterns are ignored and you'll feel a single pulse each time.
